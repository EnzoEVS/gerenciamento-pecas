# Gerenciamento de Peças

Sistema de controle de peças para oficina mecânica.

## Fluxo

O sistema funciona em etapas:

1. Marca
2. Carro/modelo
3. Ano
4. Versão
5. Peças

Exemplo:

Chevrolet → Astra → 2000 → 1.8

## Recursos

- Adicionar marcas
- Adicionar carros
- Adicionar anos
- Adicionar versões
- Remover itens
- Controle de peças

## Rodar localmente

Abra o index.html no navegador.
